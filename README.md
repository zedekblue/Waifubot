# Waifubot
 Minecraft server status for Discord Bot
   
Want to run this yourself?  
Download source  
Download npm  
in bot folder run:  
npm install discord.js  
npm install async  
npm install request  
  
create owner.json:  
{"id": "your discord id"}  
  
create auth.json:  
{"token": "your bot token"}  
  
create users.json:
{"list": []}
   
run bot with:  
node ./bot.js
